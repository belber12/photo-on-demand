# Mac Mouse Wheel Fix

Инвертирует направление скролла **только для мышки**, не трогая трекпад.

## Проблема

На Mac в настройках один переключатель управляет скроллом и мышки, и трекпада.
Если выключить "Естественный скроллинг" — исправится мышка, но сломается трекпад.
Эта программа решает проблему: перехватывает системные события скролла и переворачивает их только для мышки (дискретный скролл), трекпад остаётся с естественным направлением.

## Требования

- macOS 10.14+
- Python 3.x (уже есть на Mac)
- pyobjc (установится автоматически)

## Быстрый старт

```bash
# Установить зависимости и запустить
pip3 install pyobjc-framework-Quartz pyobjc-framework-Cocoa
python3 mouse_wheel_fix.py
```

## Автозапуск при входе в систему

```bash
chmod +x install.sh
bash install.sh
```

## Удаление автозапуска

```bash
bash uninstall.sh
```

## Разрешения

При первом запуске macOS попросит разрешить доступ:

**Системные настройки → Конфиденциальность и безопасность → Специальные возможности**

Добавь **Терминал** (или **iTerm2**) в список разрешённых приложений.

## Как это работает

Программа использует `CGEventTap` из macOS Core Graphics API для перехвата событий скролла.
Ключевое поле `kCGScrollWheelEventIsContinuous`:
- `0` = мышь (дискретный скролл) → инвертируем
- `1` = трекпад (непрерывный скролл) → не трогаем
